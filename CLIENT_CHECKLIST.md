## 1. 도메인 (people-art.co.kr)

### 도메인이 뭔데?
쉽게 말하면 웹사이트 주소야. `people-art.co.kr`이 바로 도메인이야.
카페24에서 구매하면 연간 얼마씩 내는 방식이고, 이 주소로 사람들이 사이트에 접속하게 돼.

### 사이트 주소 구성
- **일반 사용자 사이트**: `people-art.co.kr`
- **관리자 페이지**: `admin.people-art.co.kr` ← 이건 추가 비용 없이 같은 도메인으로 만들 수 있어

> 관리자 페이지 주소를 따로 사서 쓰는 게 아니라, 같은 도메인에 `admin.` 을 앞에 붙여서 쓰는 방식이야. cafe24에서 별도 구매 없이 설정만 하면 돼.

### 해줘야 할 것
- [ ] cafe24에서 `people-art.co.kr` **이미 구매했는지 확인**
- 구매 완료했으면 → cafe24 로그인 정보 알려줘 (DNS 설정 변경해야 해서)
  - 또는 내가 변경해야 할 설정값 알려주면 네가 직접 변경해도 돼
- 아직 안 샀으면 → 아래에서 구매해줘

**구매 방법:**
1. [cafe24 도메인 검색](https://www.cafe24.com/domain/) 접속
2. 검색창에 `people-art` 입력
3. `.co.kr` 선택해서 구매

## 2. 사업자 정보

### 왜 필요해?
고객한테 결제를 받으려면 법적으로 사업자 등록이 되어 있어야 해.
결제 서비스에 심사를 받을 때 이 서류들이 필요해.

### 준비해야 할 것들
- [ ] **사업자등록증** (PDF 또는 사진) — 세무서나 홈택스에서 발급 가능
- [ ] **사업자등록번호** (사업자등록증에 있는 10자리 번호)
- [ ] **대표자 이름**
- [ ] **사업장 주소**
- [ ] **정산 받을 은행 계좌** — 고객 결제금이 입금될 계좌
  - 은행명, 계좌번호, 예금주
- [ ] **통장 사본** (PDF 또는 사진)

> 아직 사업자가 없으면 먼저 사업자 등록을 해야 해. 홈택스(hometax.go.kr)에서 온라인으로 할 수 있어.

## 3. 포트원 (결제 서비스) 가입 — 제일 먼저 해!

### 포트원이 뭔데?
고객이 멤버십 결제할 때 신용카드, 카카오페이 같은 걸 처리해주는 서비스야.
우리가 직접 카드사랑 계약하는 건 복잡하니까, 포트원이 중간에서 연결해줘.
수수료는 카드 결제금액의 약 2~3% 정도야 (카드사마다 다름).

### 가입 방법

**Step 1. 포트원 회원가입**
1. [https://portone.io](https://portone.io) 접속
2. 우측 상단 **"무료로 시작하기"** 클릭
3. 이메일, 비밀번호 입력해서 가입
4. 이메일 인증 완료

**Step 2. 내 키 확인 (나한테 전달해줘야 하는 정보)**
1. 로그인 후 좌측 메뉴에서 **"연동 관리"** → **"식별코드 · API Keys"** 클릭
2. 아래 3가지 복사해서 나한테 전달:
   - `고객사 식별코드` (예: `imp12345678`)
   - `REST API Key`
   - `REST API Secret`

**Step 3. PG사 신청 (이게 심사가 걸림)**
> PG사 = 실제 카드 결제를 처리하는 회사야 (KG이니시스, NHN KCP 같은 곳)

1. 좌측 메뉴 **"결제 연동"** → **"PG사 설정"** 클릭
2. **KG이니시스** 또는 **NHN KCP** 중 하나 선택해서 신청
   - 둘 다 국내 대표 PG사야, 어느 거든 괜찮아
3. 신청 폼에 **2번에서 준비한 사업자 정보** 입력
4. ⭐ **중요**: 결제 유형 선택 시 **"정기결제(구독)"** 반드시 체크
   - 멤버십 자동 갱신 기능에 필요해
5. 제출 후 심사 대기 (2~5 영업일)

> 심사 승인 완료되면 연락이 와. 승인 후 나한테 알려줘.

## 4. 카카오 소셜 로그인 설정

### 이게 왜 필요해?
"카카오로 시작하기" 버튼 만들 때 필요해.
카카오 개발자센터에서 앱을 등록해야 허가를 받을 수 있어.

### 설정 방법

**Step 1. 카카오 개발자센터 접속**
1. [https://developers.kakao.com](https://developers.kakao.com) 접속
2. 우측 상단 **"로그인"** → 카카오 계정으로 로그인
   - 회사/서비스 대표로 쓸 카카오 계정으로 하는 게 좋아

**Step 2. 앱 만들기**
1. 상단 **"내 애플리케이션"** 클릭
2. **"애플리케이션 추가하기"** 클릭
3. 아래 내용 입력:
   - 앱 이름: `피플앤아트`
   - 사업자명: 회사/본인 이름
4. **"저장"** 클릭

**Step 3. 정보 확인 및 설정**
1. 생성된 앱 클릭
2. **"앱 키"** 탭 → `REST API 키` 복사해서 나한테 전달
3. 좌측 메뉴 **"카카오 로그인"** 클릭
4. **"활성화 설정"** → ON으로 변경
5. **"Redirect URI"** 항목에 아래 주소 추가:
   ```
   https://people-art.co.kr/auth/callback
   ```
   (개발 중에는 `http://localhost:3000/auth/callback`도 추가해줘)
6. 좌측 **"동의항목"** → 아래 항목 **필수 동의**로 설정:
   - `닉네임`
   - `카카오계정(이메일)` → **"필수 동의"** 선택
   - 이메일은 심사가 필요할 수 있어. 심사 신청 버튼 뜨면 클릭해서 제출해줘

**나한테 전달할 정보:**
- `REST API 키`

## 5. 구글 소셜 로그인 설정

### 설정 방법

**Step 1. 구글 클라우드 콘솔 접속**
1. [https://console.cloud.google.com](https://console.cloud.google.com) 접속
2. 구글 계정으로 로그인 (서비스 대표 구글 계정 권장)
3. 처음 접속하면 이용약관 동의 → 동의 후 진행

**Step 2. 프로젝트 생성**
1. 상단 중앙에 프로젝트 선택 드롭다운 클릭
2. **"새 프로젝트"** 클릭
3. 프로젝트 이름: `peopleart`
4. **"만들기"** 클릭
5. 생성 완료 후 해당 프로젝트 선택

**Step 3. OAuth 동의 화면 설정**
1. 왼쪽 메뉴 (햄버거 버튼) → **"API 및 서비스"** → **"OAuth 동의 화면"**
2. 사용자 유형: **"외부"** 선택 → **"만들기"**
3. 아래 내용 입력:
   - 앱 이름: `PEOPLE & ART`
   - 사용자 지원 이메일: 대표 이메일 주소
   - 개발자 연락처 이메일: 대표 이메일 주소
4. **"저장 후 계속"** 클릭 → 이후 화면들도 기본값으로 **"저장 후 계속"**

**Step 4. 클라이언트 ID 발급**
1. 왼쪽 메뉴 **"사용자 인증 정보"** 클릭
2. 상단 **"+ 사용자 인증 정보 만들기"** → **"OAuth 클라이언트 ID"**
3. 애플리케이션 유형: **"웹 애플리케이션"** 선택
4. 이름: `peopleart-web`
5. **"승인된 리디렉션 URI"** → **"URI 추가"** 클릭 후 입력:
   ```
   https://people-art.co.kr/auth/callback
   ```
6. **"만들기"** 클릭
7. 팝업에 표시되는 아래 정보 복사해서 나한테 전달:
   - `클라이언트 ID`
   - `클라이언트 보안 비밀번호`
   - (팝업 닫으면 보안 비밀번호 다시 확인하기 번거로우니까 바로 복사해둬!)

**나한테 전달할 정보:**
- `클라이언트 ID`
- `클라이언트 보안 비밀번호`

## 6. 이메일 발송 서비스 (Resend) 가입

### 이게 왜 필요해?
멤버십 만료 알림, 고객이 지원 기관에 이력서 제출할 때 자동 이메일 발송하는 기능에 필요해.
월 3,000통까지 무료야.

### 가입 방법

**Step 1. 회원가입**
1. [https://resend.com](https://resend.com) 접속
2. **"Sign Up"** 클릭
3. 구글 계정 또는 이메일로 가입

**Step 2. API 키 발급**
1. 로그인 후 왼쪽 메뉴 **"API Keys"** 클릭
2. **"Create API Key"** 클릭
3. Name: `peopleart` 입력 → **"Add"**
4. 생성된 키 즉시 복사! (창 닫으면 다시 못 봐)
5. 복사한 키 나한테 전달

**Step 3. 도메인 등록 (발신 이메일 설정)**
1. 왼쪽 메뉴 **"Domains"** 클릭
2. **"Add Domain"** 클릭
3. 도메인 입력: `people-art.co.kr`
4. 나타나는 DNS 레코드 값들 캡처해서 나한테 전달
   - 내가 cafe24 DNS에 설정하는 방법 알려줄게

> 이 설정 하면 `noreply@people-art.co.kr` 같은 주소로 이메일 보낼 수 있어.

**나한테 전달할 정보:**
- `API Key`
- DNS 레코드 캡처 화면
